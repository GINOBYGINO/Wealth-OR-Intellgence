# WEALTH OR IQ

> **如果出生時的財富，必須用智商來交換，你會選擇哪一種命運？**

**Birth Lottery｜命運交換所** 是一個互動式網頁實驗，
透過假設性的數學模型，模擬「出生財富百分位」與「智商（IQ）」之間的交換關係，
引導使用者思考 **階級、天賦、運氣與不平等** 等議題。

---

##  專案簡介

在本系統中，使用者可以選擇自己希望出生在「全球財富前 N%」，
但必須接受一個對應下降或上升的智商數值。

* 財富越高 → 智商越低
* 財富越低 → 智商越高

這不是科學模型，而是一個 **思想實驗**。

---

##  使用方式

1. 開啟 `index.html`
2. 輸入你希望出生在 **全球財富前 N%**
3. 系統會即時計算並顯示：

   * 對應的預測 IQ
   * 財富百分位
   * 估計的個人資產
4. 透過圖表觀察：

   * IQ 成長曲線
   * 財富分布曲線（對數座標）

---

## 核心模型說明

### 智商計算公式（假設）

```
IQ = 60 × 3^(N / 100)
```

* 當 N 越大（越貧窮）
* IQ 以指數方式成長
* 100% 對應極端高智商（理論值）

> 此公式 **非現實、非科學**，僅用於概念表達。

---

### 財富估算邏輯（近似模型）

* Top 1%：指數成長（設有上限以利視覺化）
* 1%–10%：線性下降
* 10%–50%：中產階級緩慢遞減
* 50% 以下：趨近於 0

財富單位以 **千美元（kUSD）** 顯示，
分布參考 **2025 年 UBS 全球財富報告（近似）**。

---

## 視覺化設計

* 使用 **Chart.js**
* 雙 Y 軸設計：

  * 左側：IQ（線性）
  * 右側：財富（對數）
* 紅點標示使用者選擇位置
* 黑白極簡風格，強調冷靜與距離感
